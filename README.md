## üõ†Ô∏è Installation & Setup

1.  **Clone the Repo**
    ```bash
    git clone https://github.com/YOUR_USERNAME/Nexus-Hyperion.git
    cd Nexus-Hyperion
    ```

2.  **Configure Environment**
    Create a `.env` file with your keys:
    ```bash
    cp .env.example .env
    nano .env
    ```

3.  **Run the Agent**
    ```bash
    go mod tidy
    go run main.go
    ```
